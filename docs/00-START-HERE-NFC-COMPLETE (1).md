## 🎨 How It Works (Technical)

### 1. NFC Tag URL Structure
```
https://bizbio.co.za/c/restaurant-slug?nfc=T5A3B9
                                       ↑
                                  Unique tag code
```

### 2. Data Flow
```
Customer taps table
  ↓
Phone reads NFC chip
  ↓
Browser opens URL with ?nfc= parameter
  ↓
Backend looks up: "T5A3B9 = Table 5"
  ↓
Check if Event Mode is enabled
  ↓
Return: Menu + Table fun fact + Images
  ↓
Frontend displays personalized menu
```

### 3. Event Mode
```
Restaurant dashboard → Toggle ON
  ↓
All tables instantly show event menu
  ↓
Only items marked "Available in Event Mode" display
  ↓
Wedding ends → Toggle OFF
  ↓
Back to regular menu
```

**Switch time:** 1 second vs. 30 minutes manual
