## 2. Product Offerings

### 2.1 NFC Table Stickers (Standard)

**Specifications:**
- **Material:** Waterproof vinyl with NFC chip
- **Size:** 40mm diameter (coin-sized)
- **Adhesive:** 3M industrial-strength
- **Lifespan:** 3-5 years
- **Read Range:** 1-3cm (requires tap, not scan)
- **Compatibility:** All NFC-enabled smartphones

**Mounting Options:**
- Stick on table surface
- Stick under table edge
- Stick on table number holder

**Pre-Programmed:** Yes, with restaurant's unique URL + table code

## 8. Frequently Asked Questions

### Product Questions

**Q: How long do NFC tags last?**  
A: Standard stickers: 3-5 years. Premium discs: 5-7 years. Both have warranties.

**Q: Are they waterproof?**  
A: Yes. Standard stickers handle spills. Premium discs are IP67 rated (submersion).

**Q: Can they be reprogrammed?**  
A: Yes. If you change your URL or want to reassign tables, we can reprogram for R20/tag.

**Q: Do they need batteries?**  
A: No. NFC uses power from the phone. Tags are passive (no power needed).

**Q: What phones work?**  
A: iPhone 7+ and most Android phones from 2016+. About 90% of smartphones in SA.

---

### Software Questions

**Q: Can I try Menu Pro before buying?**  
A: Yes! 14-day free trial when you purchase NFC tags (package deals include trial).

**Q: What if I only need Event Mode, not table personalities?**  
A: Menu Pro includes both. But you can use just Event Mode and ignore tables feature.

**Q: Can I have different fun facts for different events?**  
A: Not in v1. Fun facts are per table, not per event. Coming in v2.

**Q: How many events can I configure?**  
A: Unlimited. Save as many event configurations as you want.

**Q: Can tables be in multiple locations?**  
A: With Multi-Location plan, yes. Each location has its own set of tables.

---

### Technical Questions

**Q: How do I link NFC tags to tables?**  
A: In your dashboard, go to Tables → Add Table → Enter NFC code (printed on tag).

**Q: What if a tag stops working?**  
A: Contact support. Warranty covers replacement if defective.

**Q: Can customers see table info without NFC?**  
A: No. Table personalities only show when accessed via NFC code. QR codes show regular menu.

**Q: What if I rearrange tables?**  
A: In dashboard, you can reassign which NFC code belongs to which table number.

---

### Pricing Questions

**Q: Is Menu Pro required to use NFC tags?**  
A: No. Basic NFC works without Menu Pro (all tables see same menu). Menu Pro adds Event Mode + Table Personalities.

**Q: Can I cancel Menu Pro anytime?**  
A: Yes. Cancel anytime. Tags still work (basic function). Event mode and table personalities deactivate.

**Q: Is there a setup fee?**  
A: No setup fee for DIY. Assisted setup included in package. White-glove has setup included in price.

**Q: Do I pay per table?**  
A: No. Menu Pro is R149/month for unlimited tables at one location. Multi-location needs separate add-on per location.
